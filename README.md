# Geonorge.Kartkatalog

Inneholder søke-api for å finne metadata, samt nedlastingskurv. 

## Kom i gang

### Forutsetninger

- .net 10

### Oppsett

1. Klon repoet:
   ```bash
   git clone git@github.com:kartverket/Geonorge.Kartkatalog.git
   cd Geonorge.Kartkatalog
   dotnet run
   ```

2. Installer avhengigheter:
   ```bash
   yarn install
   ```

3. Konfigurer miljøvariabler — se `appsettings.json`- filene.
